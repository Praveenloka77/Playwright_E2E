# Playwright E2E Automation Framework

## Project Overview

This project is a **Playwright End-to-End Automation Framework** built using TypeScript.
It covers UI testing and real-time test scenarios.

## Tech Stack

* Playwright
* TypeScript
* Node.js

##  Project Structure

playwright-e2e-project/
 ├── tests/
 ├── pages/
 ├── utils/
 ├── playwright.config.ts
 ├── package.json


## Run Tests

```bash
npx playwright test
```

## View Report

```bash
npx playwright show-report
```

## Features

* Page Object Model (POM)
* Screenshot on failure
* Trace Viewer support
